# WordPictureAddIn

桌面版 Microsoft Word 图片双击预览插件，类似WPS的功能。

## 功能简介

- 在 Word 中双击图片，弹出全屏预览层
- 预览层支持图片缩放、拖动查看
- 支持临时标注(支持shift吸附)：
  - 直线
  - 矩形
  - 圆形
  - 箭头
  - 文字
- 支持撤销、恢复原比例、保存当前带标注的图片
- 标注内容只保留在当前预览窗口中，关闭后自动清空
